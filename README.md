# 📖 Descrição

O **Investment Dashboard** é uma aplicação web desenvolvida para apresentar e visualizar informações relacionadas a investimentos por meio de um painel interativo.

O sistema organiza dados financeiros em uma interface visual, permitindo acompanhar métricas importantes através de **gráficos, indicadores e componentes de dashboard**.

Aplicações desse tipo são amplamente utilizadas em plataformas financeiras para **monitorar portfólios de investimento, analisar desempenho de ativos e facilitar a tomada de decisão baseada em dados**.

O projeto foi desenvolvido utilizando tecnologias fundamentais da web, permitindo execução direta no navegador sem necessidade de dependências externas.

---

# 🚀 Fluxo simplificado da aplicação:

```text
Usuário acessa o dashboard
        ↓
Sistema carrega os dados de investimento
        ↓
Informações são organizadas em gráficos e indicadores
        ↓
Usuário analisa métricas financeiras no painel
```

O dashboard foi projetado para demonstrar como **dados financeiros podem ser organizados e visualizados de forma clara e interativa em aplicações web**.

---

# 🧠 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais do desenvolvimento web:

* **HTML5** — estrutura da interface do dashboard
* **CSS3** — layout e estilização dos componentes visuais
* **JavaScript (ES6)** — lógica da aplicação e manipulação de dados

Essas tecnologias permitem que o sistema funcione diretamente no navegador, tornando a aplicação leve e fácil de executar.
